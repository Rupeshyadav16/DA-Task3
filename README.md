# DA-Task3
# ApexPlanet Data Analytics Internship Portfolio - End-to-End Enterprise Data Pipeline

## 📌 Project Overview
This repository contains the complete enterprise-grade deliverables compiled during my Data Analytics track at **ApexPlanet Software Pvt. Ltd.** The project is structured across three comprehensive operational milestones, transforming raw, messy transactional records into optimized predictive intelligence and executive-ready reporting assets.

---

## 🛠️ Technology Stack & Environment
* **Core Language:** Python 3.10+
* **Data Manipulation:** `pandas`, `numpy`, `openpyxl`
* **Statistical Visualizations:** `matplotlib`, `seaborn`
* **Machine Learning & Predictive Modeling:** `scikit-learn` (Linear Regression Model)
* **Relational Database Engine:** `sqlite3` (In-Memory SQL processing architecture)
* **Automated Reporting:** `python-pptx` (Programmatic layout compilation)
* **Environment:** VS Code Terminal Infrastructure & GitHub Version Control
